# House Prices Prediction — Machine Learning Project

## Project Overview

This project builds a machine learning model to predict house sale prices using the **Kaggle House Prices dataset**.

The goal is to develop a complete supervised learning pipeline including:

* Data exploration
* Feature preprocessing
* Feature engineering
* Model training
* Model evaluation
* Model comparison

---

## Dataset

Dataset: *Kaggle House Prices – Advanced Regression Techniques*

Dataset link:
[House Prices Data Set](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data)

The dataset contains **79 explanatory variables** describing various aspects of residential homes.

Examples of feature categories include:

* Property characteristics
* Location features
* Construction details
* Interior and exterior quality
* Basement and garage features

Target variable:

`SalePrice` – the final sale price of the house.
